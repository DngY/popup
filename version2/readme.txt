
1. 说明

此popup.js原型来源于百度空间的popup.js(目前百度空间好像不用这个了)，作者根据sso系统业务需求进行了简单的修改定制。

2. 使用方法

在需要使用的页面中引入jquery.js和本popup.js，注意jquery.js引用在前，然后在页面加载的时候加入以下query代码：
$("#元素ID值").login(url地址);

3. demo

demo参看demo.html

4. 关于弹出框样式

如要修改样式需修改popup.js中的部分代码，颜色部分为以下代码：
this.color = {
			cColor:"#666", //蒙皮颜色
			bColor:"#EEEEEE", //背景颜色
			tColor:"#FFFFFF", //标题背景颜色，边框颜色
			wColor:"#000000" //标题文字颜色
		};
用户可根据自己需要对颜色进行修改，其它样式作者也在探索中，用户可看看源代码根据自己需要进行定制
